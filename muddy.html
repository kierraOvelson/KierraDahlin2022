Muddy Boots - Landscaping

With all logo designs, I go through many different sketches, this was definitely the case with Muddy Boots, a landscaping company! I was working with the instructions along the way, which isn’t always great, as towards the end the client had a different request for the logo; a badge. The colours are attractive greens, that are more on the subtle side. In the end I really like how it turned out, however I recognize that this logo isn’t ideal for all uses and not the best for scaling.
